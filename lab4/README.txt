Файлы:
1. DataAccess.cs (Класс для обработки данных из базы данных, находится в папке DataAccess)
2. DataService.cs (Windows-служба для работы с базой данных, находится в папке ServiceLayer)
3. XmlDbSerializer.cs (xml сериализатор, находится в папке DataManager)
4. AirLines.cs, Planes.cs, ServiceStaff.cs, Pilots.cs, Companies.cs (сущности базы данных, находится в папке DataModel)
5. Base.png (Фотка диаграммы базы данных)
 
При запуске службы(В методе OnStart):
1. Достаются значения полей классов из базы данных в экземпляры класса контейнера.
2. Список из класса контейнера сериализуется в xml файл.
3. При возникновении ошибок они записываются в Error_Log.txt.
